Download do Cliente Interlude

Para conseguir jogar o Lineage Brasil Club, caso você não possua, você precisará baixar e instalar o cliente do Lineage na versão interlude. O processo de instalação é simples e intuitívo. Siga o passo a passo e você conseguirá instalar tranquilamente.

Tendo feito esse caminho, você perceberá que existe uma pasta chamada system. Basta excluir essa pasta e adicionar ao lugar dela a pasta system que será baixada no link a seguir:

Links Para Baixar o Cliente

  • Opção 1: https://gameborder.net/file/lineage-ii-interlude-client
  • Opção 2: https://gamesnowcliente.com/line-2-interlude

Após baixar e instalar o cliente, você precisará substituir a pasta system pela system disponibilizada pelo Lineage Brasil Club.
Clique aqui para baixar e instalar a System do Lineage Brasil Club e poder jogar.
